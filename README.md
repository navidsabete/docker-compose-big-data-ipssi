# docker-compose-big-data-ipssi
DOCKER COMPOSE BIG DATA - IPSSI

docker compose jupyter avec pyspark pour prép à la suite de l'exo
makefile pour faciliter la commande

build & run the container : afficher les logs pour récupérer le token
poour faciliter (comme indiqué sur la page d'accueil de jupyter) mettre un password

copy paste token et mettre  un nouveau password sur le formulaire en bas de la page
quand besoin de login, juste mettre le password
